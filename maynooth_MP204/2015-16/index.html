Study recommendations for exam preparation

In decreasing order of importance (most important listed first):

(1) Work through the assignment sets, all 11 of them. Try to ensure that you are able to do each problem without having to look up anything.

(2) Look through the material covered in class, and try to make sure you are able to reproduce each derivation or argument.

(3) Work through some past MP204 exams, which you can obtain through the library website. Also work through the sample exam posted above.

(4) Work through Charles Nash's notes, which is linked to below on this page. Most (but not all) of our material is contained in these notes.

In the unlikely case that you have time to do more than the above ---

(5) Work through the Feynman lectures (Volume II), which are free to read on this website. The material we have covered is mostly contained within the first 20 chapters of Volume II.
(Specifically: Chapters 1, 4--6, 13--18, 20.)

(6) Work with another textbook, for example Introduction to Electrodynamics, by D.J. Griffiths. The university library has several copies. If you work through some examples in each of the relevant chapters with pen and paper, you will probably get good additional preparation.

Notation

We will use SI (also called MKS or MKSA) units. Note that many equations look quite different when written in Gaussian (or CGS) units. When reading a textbook, be sure to watch out for which units that text is using.

Notations vary. I will mostly try using the same notations as in Prof. Nash's notes, but will not always succeed. You anyway need to be able to read and learn from multiple sources using different notations for the same physical quantities.
